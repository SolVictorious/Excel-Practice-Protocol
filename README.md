# Excel-Practice-Protocol
A simple practice demonstration for having a messy data set and proper cleaning and analysis using Excel

Here, I was given a specific messy data set generated by anthropic's Claude for a fictional organization. 
The aim is to clean up the dataset, find insights, and answer 6 questions. 

1-What's our actual net revenue for the period? 
2-Which region is most profitable? 
3-Is a rep over-discounting?
4-Is there a product category quietly losing us money?
5-Enterprise vs. Consumer, which is actually worth more to us?
6-Suspicions of narratives that state growth from Jan to Jun, Confirmation or refutation?
